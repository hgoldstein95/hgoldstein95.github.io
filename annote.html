an\note

an\note is a web-based application designed to make taking notes (generally at the High School or University level) more effective. Users usespecial an\note tags within their notes, and the notes will be automatically formatted and categorized for easy reviweing and studying later.

The application runs on a node.js server, and interfaces with mongoDB database that stores saved notes. The interpreter which formats the notes and handles an\note tags is built in Javascript with jQuery, and is based on Markdown.

The team worked on this project at Big Red Hacks in the Fall of 2014 at Cornell University, so it is somewhat unpolished. They hope to eventually add a user-friendly account system and saved notebooks, allowing the app to be fully functional for end-users.
